# Instruction Manual
## Scraping & Parsing
### First-time running
In the CLI, type `chmod +x interface.sh parse.sh web-scraper.sh`.
### How to run the program
In the CLI, type `./interface.sh`

It is recommended to choose option 1 from the menu.

The list of faculty & staff can be found in the `data` folder under `names.txt`.
